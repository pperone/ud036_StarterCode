# ud036_StarterCode
Source code for a Movie Trailer website.

## Installation
In order to run this code, Python 2.7 must be installed in your machine. If that is not the case, you can download it [here](https://www.python.org/downloads/).
To run the code you can either open and execute it from the Python IDLE or, from command prompt or terminal, navigate to the directory where the files are and type:
`python entertainment_center.py`
